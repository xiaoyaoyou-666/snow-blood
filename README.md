# 🎯 太行残雪 | 视觉小说项目

## 📝 项目简介
本项目是软件工程团队打造的红色主题沉浸式剧情向视觉小说，以太行山区抗战与革命历史为背景，以小人物的命运为切口，深度影响，以轻量化、高沉浸的方式完成革命历史背景下的情感叙事。

项目以公益+文化传播为导向，旨在为博物馆、红色教育基地打造创新型红色文化体验载体，用细腻的剧情与美术搭建一座触摸历史温度的桥梁。

## 👥 目标用户
- 青少年与学生群体
- 博物馆/红色基地访客
- 红色文化爱好者
- 红色教育机构/学校

## ✨ 核心功能模块及亮点
### 功能模块描述
- **交互式剧情分支**：关键节点提供多选项，影响剧情走向、角色好感度与结局解锁
- **多结局收集系统**：提供主线、支线、隐藏等多种结局，支持结局图鉴查看与回顾
- **存档与读档系统**：支持存档，可随时回到关键分支点重新选择
- **文本自动播放与回看**：支持自动播放文本、调整播放速度，可回看历史对话梳理剧情

### 创新点描述
- **选择驱动式叙事**：市面多数视觉小说以单线剧情为主，本项目实现“选择→剧情→结局”的完整闭环，玩家的每一个决定都将影响角色命运与故事走向。
- **历史场景沉浸还原**：针对民国太行山区的时代背景定制场景与对话，解决通用剧情游戏背景悬浮的问题，更贴合真实历史氛围与用户情感体验。
- **轻量化跨平台适配**：以Web端/轻量客户端为载体，无需复杂配置即可运行，降低玩家体验门槛，让更多人能接触到叙事游戏的魅力。

## 👥 成员分工（团队一共2人，每一件事情都由两人合作完成）
- **剧情策划组（2人）**：完成世界观搭建、剧情撰写、分支逻辑与多结局设计
- **美术设计组（2人）**：完成角色立绘、场景CG、UI界面设计与美化
- **程序开发组（2人）**：实现剧情引擎、存档系统、分支逻辑与画廊功能开发
- **统筹总结组（2人）**：梳理项目整体内容，完成文稿汇总、测试优化与成果汇报

## 🛠️ 开发与运行环境
- 编程语言：Java >= 11
- 核心环境：
  - Spring Boot >= 2.7
  - JavaFX >= 17
  - MySQL >= 8.0
- 适配场景：PC端桌面应用运行、场馆大屏互动模式、轻量Web端嵌入
